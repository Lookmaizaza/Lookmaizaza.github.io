มีบางครั้งที่ข้อมูลไม่ได้บอกเราเพียงแค่ว่า “อะไรเกิดขึ้น” แต่กลับทำหน้าที่เป็นเสมือนกระจกสะท้อนให้เห็นว่า “คนคิดอย่างไรและตัดสินใจอย่างไร” 

การวิเคราะห์ข้อมูล **"ร้อยละของประชากรที่เดินทางท่องเที่ยว จำแนกตามพาหนะที่ใช้ในการเดินทาง ภาคที่อยู่อาศัย และลักษณะการเดินทาง"** จากสำนักงานสถิติแห่งชาติ ในครั้งนี้ ไม่ได้เริ่มต้นจากการตั้งคำถามว่า “จะใช้กราฟอะไรให้สวยงาม” แต่เริ่มต้นจากคำถามว่า *"หากจำเป็นต้องออกแบบโครงสร้างพื้นฐานหรือบริการด้านการท่องเที่ยว ปัจจัยใดคือแกนกลางของพฤติกรรมผู้บริโภค?"*

งานนี้บูรณาการแนวคิดจากหนังสือ **"ภาพนิทัศน์จากข้อมูลในฐานะศิลปะเพื่อการใช้ประโยชน์"** ซึ่งระบุอย่างชัดเจนว่า ภาพนิทัศน์ที่เป็นศิลปะเพื่อการใช้งานนั้น มีภารกิจสำคัญในการให้ความรู้ สติ ปัญญาแก่สังคม หรือให้ความสว่างทางปัญญาแก่ผู้ชม ผนวกกับการเชื่อมโยงความรู้เชิงปฏิบัติการจาก:
* **11.1.6 Lab – Measures of Central Tendency:** การวิเคราะห์ข้อมูลด้วยสถิติพื้นฐาน (Mean, Median, Mode) เพื่อพิสูจน์ความเสถียรของพฤติกรรม

## 1. กรอบแนวคิดในการวิเคราะห์ (Analytical Framework)

การวิเคราะห์นี้ดำเนินตามกรอบ **Data → Information → Insight → Action** โดยตั้งเป้าหมายตามหลักการของ Ad-Hoc Report ว่า *"ถ้าผู้บริหารมีคำถามทางธุรกิจตอนนี้ ข้อมูลชุดนี้ต้องตอบได้ทันที"*

ตัวแปรหลักจากชุดข้อมูลที่นำมาใช้ในการออกแบบการเล่าเรื่อง มีดังนี้:

| ตัวแปรหลัก (Variables) | Field ในฐานข้อมูล | บทบาทในการเล่าเรื่อง (Storytelling Role) |
| :--- | :--- | :--- |
| **พาหนะ** | `vehicle_used` | ตัวแทนของ "ทรัพยากรที่ประชากรมี" และความพึงพอใจหลัก |
| **ภูมิภาค** | `region` | ตัวแทนของ "บริบททางเศรษฐกิจและข้อจำกัดเชิงพื้นที่" |
| **ลักษณะทริป** | `type_travel` | ตัวแทนของ "เงื่อนไขที่มีผลต่อการตัดสินใจ" |

## 2. การประยุกต์ใช้ Lab 11.1.6 กับข้อมูลจริง (Mean / Median / Mode)

หลักการสำคัญประการหนึ่งจาก Lab คือ ค่าเฉลี่ย (Mean) ที่ดีต้องไม่ถูกบิดเบือนด้วยค่าผิดปกติ (Outlier) และมัธยฐาน/ฐานนิยม (Median/Mode) ต้องทำหน้าที่ช่วยยืนยันพฤติกรรมที่เกิดขึ้นซ้ำ เมื่อนำมาคำนวณกับตัวแปร `vehicle_used` พบผลลัพธ์ที่น่าสนใจ ดังนี้:

| พาหนะ | Mean (%) | Median (%) | Mode (%) | ความหมายเชิงพฤติกรรม |
| :--- | :--- | :--- | :--- | :--- |
| **รถยนต์ส่วนตัว** | ~65–70 | ใกล้เคียง Mean | เกิดซ้ำบ่อยสุด | พฤติกรรมกระแสหลักของประเทศ |
| **รถทัวร์/รถโดยสาร** | ~18 | คงที่ | คงที่ | ทางเลือกสำรองอันดับหนึ่ง |
| **รถเช่า** | ~13.6 | คงที่ | คงที่ | ทางเลือกที่มีนัยสำคัญเชิงพื้นที่ |
| **เครื่องบิน** | ~2-3 | ต่ำมาก | ต่ำมาก | ทางเลือกที่ถูกใช้เฉพาะเจาะจงตามเงื่อนไข |

**ข้อค้นพบสำคัญ:** ค่า Mean ≈ Median ≈ Mode ของ "รถยนต์ส่วนตัว" มีความใกล้เคียงกันมากและเกาะกลุ่มในระดับสูง สิ่งนี้เป็นเครื่องยืนยันทางสถิติว่า การใช้รถยนต์ส่วนตัวไม่ใช่ค่าบังเอิญ แต่เป็นพฤติกรรมที่มีความเสถียร (Stability) อย่างยิ่ง

## 3. ข้อค้นพบเชิงประจักษ์ (Empirical Insights)

### Insight 1: คนไทย “ขับรถไปเที่ยว” เป็นหลัก
รถยนต์ส่วนตัวคือตัวเลือกเริ่มต้น (Default Behavior) ของคนไทย การวิเคราะห์นี้พบว่าพาหนะส่วนตัวมีสัดส่วนการใช้งานเฉลี่ยสูงถึงร้อยละ 65.7 ในระดับประเทศ (ข้อมูลรวม ปี 2560) ในขณะที่รถโดยสารประจำทาง/รถตู้มีสัดส่วนเพียงร้อยละ 16.4 สอดคล้องกับแนวคิดการออกแบบภาพนิทัศน์ที่ว่า ต้องทำให้สิ่งที่ครอบงำข้อมูลโดดเด่นและเห็นได้ชัดเจนที่สุด ดังนั้น ในการนำเสนอ เราจึงต้องทำให้สัดส่วนของรถยนต์ส่วนตัวเด่นตระหง่านตั้งแต่กราฟแรก เพื่อดึงความสนใจไปสู่พฤติกรรมหลักของประเทศ

### Insight 2: บริบทเชิงพื้นที่และโอกาสของธุรกิจ "รถเช่า"
เมื่อเจาะลึกเฉพาะกลุ่มผู้ใช้รถยนต์ส่วนตัวและเปรียบเทียบตาม `region` พบว่า **ภาคตะวันออกเฉียงเหนือ (อีสาน)** มีสัดส่วนการใช้รถส่วนตัวต่ำกว่าภาคอื่นอย่างชัดเจน แต่กลับมีสัดส่วนการใช้ "รถเช่า" ที่สูงขึ้นอย่างมีนัยสำคัญ โดยกรุงเทพมหานครมีอัตราการใช้รถเช่าเพียงร้อยละ 6.8 แต่ภาคตะวันออกเฉียงเหนือและภาคเหนือกลับมีสัดส่วนการใช้รถเช่าสูงถึงร้อยละ 21.5 และ 18.3 ตามลำดับ (ปี 2560) 

สิ่งนี้สะท้อนหลักการออกแบบจากหนังสือที่ว่า *"จงทำให้มั่นใจว่าข้อมูลนั้นเด่นชัดและทำให้มีความแตกต่างกัน"* ค่าเฉลี่ยระดับประเทศมักซ่อนความจริงบางอย่างไว้ แต่การแตกข้อมูลตามพื้นที่ทำให้เรามองเห็นโครงสร้างทางเศรษฐกิจ การเข้าถึงทรัพยากร และรูปแบบการท่องเที่ยวแบบหมู่คณะที่แตกต่างออกไป

### Insight 3: การเปลี่ยนพาหนะตามเงื่อนไขการเดินทาง
เมื่อวิเคราะห์ตัวแปร `type_travel` พบพฤติกรรมที่มีความเป็นเหตุเป็นผลสูง:
* **ไม่ค้างคืน (ไป-กลับ):** สัดส่วนการใช้เครื่องบินทั่วประเทศแทบจะเป็นศูนย์ (ร้อยละ 0.1 ในปี 2560) 
* **พักค้างคืน:** สัดส่วนการใช้เครื่องบินพุ่งสูงขึ้นอย่างชัดเจน โดยเฉพาะจุดหมาย/ต้นทางอย่าง กทม. (ร้อยละ 8.3) และภาคใต้ (ร้อยละ 9.6) 

นี่คือตัวอย่างที่ชัดเจนของการดึงข้อมูลมาตอบคำถามตามหลัก Ad-Hoc Report ที่ว่า *"ถ้ามีคำถามธุรกิจ ข้อมูลต้องตอบได้ทันทีและตรงประเด็น"*

## 4. การออกแบบ Dashboard ตามหลักทฤษฎีภาพนิทัศน์

ในการออกแบบ Dashboard ไม่ได้เริ่มต้นจากคำถามว่า "อยากใช้กราฟอะไรให้ดูทันสมัย" แต่เริ่มต้นจาก **"อยากให้คนดูเข้าใจตรรกะอะไร"** โดยยึดหลักการโน้มน้าวผู้อ่านด้วยการอธิบายทางสายตา (Visual explanation) และให้ข้อมูลพูดด้วยตนเอง ผ่านการจับคู่ตัวแปรดังนี้:

| องค์ประกอบกราฟิก (Chart) | การจับคู่ข้อมูล (Data Mapping) | เป้าหมายการสื่อความหมาย |
| :--- | :--- | :--- |
| **Bar Chart** | `vehicle_used` vs `per_travel` | เพื่อให้เห็นการครอบงำ (Dominance) ของพาหนะส่วนตัวในระดับภาพรวมทันที |
| **Bar Chart แยกภูมิภาค** | `region` *(Filter: เฉพาะรถส่วนตัว/รถเช่า)* | เพื่อดึงความแตกต่างเชิงพื้นที่ โดยเฉพาะภาคอีสาน ให้โดดเด่นออกมา |
| **Slopegraph / Grouped Bar** | `type_travel` + `vehicle_used` *(เครื่องบิน)* | เพื่อเน้นให้เห็นอัตราการเปลี่ยนพฤติกรรมที่แปรผันตามเงื่อนไขของระยะเวลาทริป |
| **KPI Cards** | ค่า Mean / Median / Mode | ทำหน้าที่ตรวจทาน (Sanity Check) เพื่อยืนยันความเสถียรของชุดข้อมูล |

**ทุกการจับคู่ Field เข้ากับ Property ของกราฟ มีเหตุผลเชิงการสื่อความหมายรองรับเสมอ**

## 5. เชื่อมกลับไปที่ Lab 16.2.4 (Ad-Hoc Thinking & Action)

การออกแบบที่เน้นฟังก์ชันเหนือความสวยงามเพียงอย่างเดียว ทำให้ Dashboard นี้สามารถแปรสภาพเป็นเครื่องมือช่วยตัดสินใจ (Decision-Support Tool) ได้ทันที:

* **ควรลงทุนโครงสร้างพื้นฐานอะไรเป็นอันดับแรก?** → ข้อมูลชี้ชัดว่าควรเทน้ำหนักไปที่โครงข่ายถนนและพื้นที่จอดรถ
* **ธุรกิจรถเช่าควรขยายตลาดที่พื้นที่ใด?** → พื้นที่ภาคเหนือและภาคอีสานมี Demand เชิงกลุ่มที่แข็งแกร่ง
* **ควรใช้งบโปรโมตตั๋วเครื่องบินกับกลุ่มเป้าหมายใด?** → มุ่งเป้าไปที่นักท่องเที่ยวที่วางแผนพักค้างคืนในพื้นที่ภาคใต้และ กทม. เท่านั้น

การวิเคราะห์ข้อมูลชุดนี้เป็นเครื่องพิสูจน์แนวคิดจากหนังสือได้อย่างเป็นรูปธรรมว่า การวาดภาพนิทัศน์จากข้อมูลและการเล่าเรื่อง เป็นทักษะที่มีความสำคัญยิ่ง เพราะจำเป็นต้องขายความคิด ให้ได้รับการยอมรับและนำไปปฏิบัติจนเกิดผลในท้ายที่สุด 

และเป็นเครื่องยืนยันในสิ่งที่ได้ฝึกฝนจาก Lab ว่า: > **สถิติพื้นฐาน + การตั้งคำถามเชิงธุรกิจที่ถูกต้อง + เครื่องมือที่เหมาะสม = Insight ที่สามารถนำไปใช้ขับเคลื่อนกลยุทธ์ได้จริง**
