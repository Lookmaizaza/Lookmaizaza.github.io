# 11.1.6 Lab: Using the Measures of Central Tendency

## 🎯 วัตถุประสงค์ของแล็บ (Lab Objective)
ฝึกฝนการใช้โปรแกรม Spreadsheet (Excel) เพื่อคำนวณหาค่ากลางของข้อมูล (Measures of Central Tendency) ได้แก่ ค่าเฉลี่ย (Mean), มัธยฐาน (Median) และฐานนิยม (Mode) โดยนำมาใช้วิเคราะห์และเปรียบเทียบคะแนนสอบของนักเรียนสองกลุ่ม (กลุ่มที่เข้าเรียน Test Prep และกลุ่มที่ไม่ได้เรียน)

## 🛠️ สิ่งที่ได้ลงมือทำ (What I Did)
* **รวบรวมและกำหนดช่วงข้อมูล:** จัดการข้อมูลคะแนนสอบจากตารางดิบที่มีตัวแปรจัดกลุ่ม (Group A/B และ Test Prep Yes/No)
* **เขียนสูตรคำนวณทางสถิติ:** * ใช้ `=AVERAGE()` และ `=AVERAGEIF()` เพื่อหาค่าเฉลี่ยรวมและค่าเฉลี่ยแบบแยกกลุ่มตามเงื่อนไข
  * ใช้ `=MEDIAN()` โดยอ้างอิงช่วงเซลล์ (Range) ที่กำหนดเฉพาะกลุ่มเพื่อหาค่ากึ่งกลางของชุดข้อมูล
  * ใช้ `=MODE.SNGL()` เพื่อหาคะแนนที่มีนักเรียนสอบได้ซ้ำกันมากที่สุด
* **สร้างตารางสรุปผล:** กรอกข้อมูลผลลัพธ์ลงใน Summary Table เพื่อให้อยู่ในรูปแบบที่พร้อมนำเสนอ

## 🧠 สิ่งที่ได้เรียนรู้ (Detailed Learnings)
การทำแล็บนี้ช่วยเสริมสร้างความเข้าใจตาม Exam Objectives ของ CompTIA Data+ ดังนี้:

* **การสำรวจข้อมูลเชิงลึก (Data Exploration - Obj 2.2):**
  * ได้เรียนรู้ว่าไม่ควรพึ่งพาแค่ "ค่าเฉลี่ย" เพียงอย่างเดียว เพราะค่าเฉลี่ยมีความอ่อนไหวต่อค่าผิดปกติ (Outliers) 
  * การคำนวณ "มัธยฐาน" ควบคู่ไปด้วย ช่วยให้เห็นภาพรวมที่แท้จริงของการกระจายตัวของข้อมูล (Data Distribution)
* **การวิเคราะห์แบบมีเงื่อนไข (Conditional Analysis):**
  * เข้าใจประโยชน์ของการใช้สูตรมีเงื่อนไข (เช่น `AVERAGEIF`) ซึ่งช่วยลดระยะเวลาในการทำงาน ไม่ต้องเสียเวลาสร้าง Filter หรือตารางแยกใหม่
  * สามารถสรุปผลเชิงเปรียบเทียบได้ทันที (Insight: นักเรียนที่เรียน Test Prep มีแนวโน้มได้คะแนนสูงกว่าอย่างชัดเจน)
* **การสื่อสารข้อมูล (Communication Approach - Obj 3.1):**
  * เห็นความสำคัญของการจัดระเบียบตัวเลขที่กระจัดกระจาย ให้อยู่ในตารางสรุปผลที่ดูง่าย ซึ่งช่วยให้ Stakeholder เข้าใจประเด็นสำคัญได้ทันที
* **ความแม่นยำในการใช้เครื่องมือ (System Precision):**
  * ได้ฝึกความรอบคอบในการกำหนดอาร์กิวเมนต์ของฟังก์ชันใน Excel (Syntax) การระบุพิกัดเซลล์ และการพิมพ์สูตรที่ต้องมีความถูกต้องแบบ 100%
